# Dot Matrix

## About the game
Dot Matrix is an incredmental game about dots. Place producers on the board to print dots. Then add consumers where dots are to consume them. Every dot consumed adds to your dot count, allowing you to buy more producers and consumers. Or you can upgrade your existing producers and consumers.

There isn't really much of a point to the game. Number go up. Pretty dots flashing on the screen. If you're really creative, you can probably make some cool patterns. Have at it!

Play it here: https://mikosama326.github.io/dot-matrix-game/

## Technical info

This repo is built using npm and vite. Install Node.js + npm, then build the code.

### Install dependencies:
```npm install```

### Run a local dev server:
```npm run dev```

The site will be hosted at http://localhost:5173/ by default. Or Vite will print the URL itself.

### To create a production build:
```npm run build```

Files will be built to the `docs/` folder

### Hosting

This game is fully client-side. No backend server required.
